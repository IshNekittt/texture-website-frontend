# Pixel Textures - Frontend

Це клієнтська частина веб-додатку "Pixel Textures", створена з використанням React та Vite. Додаток є каталогом цифрових текстур з можливістю фільтрації, сортування та адміністрування.

## Основні технології

- **Фреймворк**: React
- **Збирач**: Vite
- **Керування станом**: Redux Toolkit
- **Маршрутизація**: React Router DOM
- **HTTP-клієнт**: Axios
- **Стилізація**: CSS Modules (без CSS-фреймворків)
- **Анімації**: Framer Motion
- **Форми**: React Hook Form

## Початок роботи

### 1. Клонування репозиторію

```bash
git clone texture-website-frontend
cd pixel-textures-frontend
```

### 2. Встановлення залежностей

Переконайтеся, що у вас встановлено Node.js (версія 18 або вище).

```bash
npm install
```

### 3. Налаштування змінних оточення

Створіть файл `.env` у кореневій папці проєкту та додайте до нього змінну, що вказує на ваш бекенд-сервер.

```dotenv
VITE_API_URL=http://localhost:3000
```

- Для локальної розробки використовуйте `http://localhost:3000`.
- Для продакшену вкажіть URL вашого розгорнутого бекенд-сервісу (наприклад, `https://some-url.com`).

### 4. Запуск проєкту

Для запуску в режимі розробки виконайте:

```bash
npm run dev
```

Додаток буде доступний за адресою `http://localhost:5173`.

## Скрипти

- `npm run dev`: Запускає сервер для розробки.
- `npm run build`: Збирає проєкт для продакшену.
- `npm run lint`: Запускає перевірку коду за допомогою ESLint.
- `npm run preview`: Запускає локальний сервер для попереднього перегляду продакшен-збірки.

## Структура проєкту

- **`public/`**: Статичні активи (іконки, зображення).
- **`src/`**: Основна папка з вихідним кодом.
  - **`api/`**: Налаштування клієнта Axios для взаємодії з бекендом.
  - **`components/`**: Перевикористовувані React-компоненти (Header, Loader, TextureCard тощо).
  - **`contexts/`**: React Context для глобального стану, не пов'язаного з Redux.
  - **`pages/`**: Компоненти-сторінки, що відповідають маршрутам (HomePage, CataloguePage, AdminLoginPage тощо).
  - **`redux/`**: Вся логіка керування станом за допомогою Redux Toolkit.
    - **`admin/`**, **`categories/`**, **`textures/`**, **`theme/`**: "Слайси" (slices) стану, розділені за сутностями.
  - **`utils/`**: Допоміжні функції (форматтери, парсери).
  - **`validation/`**: Схеми валідації (наприклад, для форм).
